## Tech News Hub
Tech News Hub is a simple and responsive web application that displays the latest technology news articles using a public news API. The website allows users to stay updated with trending tech headlines, search for specific topics, and explore articles in a clean and user-friendly interface.

## Project Purpose
The main purpose of this project is to build a beginner-friendly web application that helps users read the latest technology news from one place.

This project is created to practice and improve important frontend development skills such as:

Working with a public API
Fetching and displaying live data
DOM manipulation
Search, filtering, and sorting features
Responsive web design
It is a practical and small project that is suitable for learning how real-world websites handle dynamic content.

## API Used
This project uses a public news API to fetch real-time technology news articles.

## Planned Features
•	Display the latest technology news articles
•	Show article title, image, source name, and short description
•	“Read More” button to open the full article
•	Search functionality to find news by keyword (for example: AI, Apple, Google, Startups)
•	Filter news based on topics or categories
•	Sort articles by newest first
•	Responsive design for desktop and mobile devices
•	Loading indicator while data is being fetched
•	Error handling if API fails or no results are found
## Technologies Used
•	HTML5 – for the structure of the website
•	CSS3 – for styling and responsive layout
•	JavaScript (Vanilla JS) – for fetching API data, DOM manipulation, and interactivity
•	GNews API – for live technology news data
